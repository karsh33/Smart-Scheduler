Smart Scheduler is a comprehensive meeting room management system designed to optimize resource allocation and streamline meeting scheduling processes. The application features a Flutter-based mobile interface, Node.js backend API, PostgreSQL database, and Python-based optimization engine. It provides intelligent resource allocation based on priority and capacity, supports multiple authentication methods including Google Sign-In, and enforces meeting ownership controls to ensure data security. The system demonstrates practical application of modern software engineering principles including microservices architecture, RESTful API design, JWT authentication, and algorithmic optimization. This project provided hands-on experience with full-stack development, cloud authentication integration, and solving real-world scheduling conflicts through algorithmic approaches.
